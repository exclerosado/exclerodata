ExcleroData

Projeto de visualização de dados

Servindo de portfólio, o projeto ExcleroData tem o intuito de colocar em prática técnicas de visualização de dados de forma simples, objetiva e com visual limpo. Sabe-se que a área de dados envolve diversas etapas que são cruciais para um bom projeto, como a coleta, transformação e avaliação. Tendo isso em mente, a visualização dos dados é de suma importância para conduzir o espectador durante toda a apresentação, de forma que a mensagem seja transmitida de forma clara, intuitiva e objetiva.

Passado o breve momento de apresentação e objetivo deste portfólio, a primeira base de dados escolhida para ser trabalhada foi extraída do site oficial da NASA, onde o tema abordado é o de mensuração da temperatura global desde 1880.

média global base

Nessa primeira visualização nos deparamos com a variação da temperatura da superfície terrestre ano após ano. O interessante desse gráfico é repararmos que as cores escolhidas auxiliam no entendimento da elevação de temperatura no decorrer da série temporal, onde preferi destacar as variações abaixo da média com uma cor mais fria, enquanto que as variações acima da média foram destacadas com uma cor que remete a uma alta temperatura.

Conforme destacado acima, a média global era de 14˚ celsius. Maiores informações aqui.

No segundo quadro, trago uma abordagem mais simples e direta, exibindo a variação da temperatura terrestre total aos longo dos anos.

Passe o mouse nos pontos para exibir o valor de cada ano.

Por último, destaquei a média de temperatura a cada década utilizando o mesmo conceito do primeiro quadro, mas desta vez utilizando um outro modelo de gráfico.

Vale destaque para a última barra que registra a maior média, sendo que é o único período que ainda não completou os 10 anos, o que traz um grande sinal de alerta para esse tema.
